# Lab 6 — Networking & Firewall

## Objective
Configure multiple NICs and secure network access using firewalld.

## Steps
1. Configure ens33 static IP
2. Configure ens34 secondary interface
3. Assign firewall zones
4. Enable firewalld
5. Add SSH service
6. Configure rich rules
7. Capture traffic using tcpdump

## Validation
- Interfaces up
- Firewall active
- SSH accessible only from allowed IP
