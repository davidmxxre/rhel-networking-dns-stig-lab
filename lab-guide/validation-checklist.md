# Validation Checklist

## Networking
- [ ] ens33 static IP configured
- [ ] ens34 configured
- [ ] Routing verified

## Firewall
- [ ] firewalld enabled
- [ ] zones assigned correctly
- [ ] rich rules applied

## DNS
- [ ] dnsmasq running
- [ ] resolv.conf uses 127.0.0.1
- [ ] hostname resolves

## Traffic
- [ ] tcpdump captures packets
